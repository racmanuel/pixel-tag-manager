== Description ==
This plugin provides an easy way to add Pixel tracking code to your site.

== Installation ==
To install this plugin follow the following steps: 
1.- download and and activate the plugin.
2.- Go to your dashboard.
3.- In the settings section of your dashboard at the end you will find the option of pixel.
4.- Click on the Pixel option.
5.- Insert your Pixel Identifier.
6.- Click on save and that\'s it!.
